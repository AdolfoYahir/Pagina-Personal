HTML

HTML son las siglas de HyperText Markup Language, que puede traducirse como lenguaje de marcas o marcado de hipertexto. El lenguaje HTML se emplea para crear las paginas web. Es muy facil ver el codigo HTML de una pagina web, la opcion exacta cambia de un navegador a otro y tambien puede cambiar de una version a otra de un mismo navegador, pero suelen tener un nombre similar.

Historia de HTML:

Los inicios del lenguaje HTML se remontan al año 1990, cuando Tim Berners-Lee creo la primera pagina web. Tim Berners-Lee definio la primera version de HTML en el año 1991. En la actualidad, la ultima version de HTML es HTML5.

Estructura basica de un documento HTML:
  • <!DOCTYPE html> - Declaracion del tipo de documento
  • <html> - Elemento raiz
  • <head> - Metadatos y enlaces a recursos
  • <body> - Contenido visible de la pagina
  • <style> - Estilos CSS internos
<!DOCTYPE html>
<html>
<head>
  <title>Mi Primera Página Web</title>
</head>
<body>
  <h1>Hola Mundo</h1>
  <p>Mi primera página web.</p>
</body>
</html>
Elementos HTML basicos:
  • <h1> a <h6> para encabezados
  • <p> para párrafos
  • <a> para enlaces
  • <img> para imágenes
  • <div> para divisiones

CSS

CSS son las siglas de Cascading Style Sheets (Hojas de Estilo en Cascada). Es un lenguaje utilizado para describir la presentacion de documentos HTML, controlando aspectos como colores, fuentes, margenes, tamaños y disposición de los elementos en la pagina web.

Historia de CSS:

CSS fue propuesto por primera vez en 1994 por Håkon Wium Lie. La primera version oficial, CSS1, se publico en 1996. A lo largo del tiempo, se han desarrollado nuevas versiones como CSS2 y CSS3, que agregan mas caracteristicas y flexibilidad en el diseño web.

Estructura basica de CSS:

La estructura de una regla CSS se compone de tres partes principales:

  • Selector: Indica a que elemento HTML se aplicaran los estilos.
  • Propiedad: Define el aspecto o caracteristica que se desea cambiar (color, tamaño, etc.).
  • Valor: Especifica el valor que tendra la propiedad.
Ejemplo de codigo CSS:

Este ejemplo se puede poner en style, o en un archivo .css y en el archivo html se le hace una referencia:

h1 {
  color: blue;
  text-align: center;
  font-family: 'Press Start 2P', cursive;
}

body {
  background-color: #f0f0f0;
  margin: 20px;
}
Propiedades CSS comunes:
  • color - Define el color del texto
  • background-color - Define el color de fondo
  • font-size - Tamaño de la fuente
  • margin - Espacio exterior de un elemento
  • padding - Espacio interior del contenido

JavaScript

JavaScript es un lenguaje de programación interpretado y orientado a objetos, utilizado principalmente para crear interactividad en las páginas web. Permite manipular elementos HTML, reaccionar a eventos del usuario y comunicarse con servidores.

Historia de JavaScript:

JavaScript fue creado en 1995 por Brendan Eich mientras trabajaba en Netscape. Originalmente se llamaba “Mocha”, luego “LiveScript”, y finalmente “JavaScript”. Desde entonces, ha evolucionado a través de estándares ECMAScript (ES), siendo ES6 (2015) una de las versiones más importantes.

Estructura básica de JavaScript:

Un programa en JavaScript se compone de instrucciones que pueden declarar variables, realizar operaciones y controlar el flujo del programa. Sus elementos principales son:

  • Variables: Almacenan datos (usando let, const o var).
  • Funciones: Agrupan código reutilizable para realizar tareas específicas.
  • Eventos: Responden a acciones del usuario (clics, teclas, etc.).
  • Condiciones y bucles: Permiten tomar decisiones y repetir tareas.
Ejemplo de código JavaScript:

Este ejemplo muestra cómo mostrar un mensaje en la consola y en el documento HTML:

// Mostrar un mensaje en la consola
console.log("¡Hola, mundo!");

// Mostrar un mensaje en la página
document.write("Bienvenido a JavaScript");
Funciones comunes de JavaScript:
  • alert() - Muestra un mensaje en una ventana emergente.
  • prompt() - Pide al usuario que ingrese un valor.
  • console.log() - Imprime mensajes en la consola del navegador.
  • document.getElementById() - Accede a un elemento HTML por su ID.
  • addEventListener() - Escucha eventos como clics o teclas.

Si deseas aprender más sobre JavaScript con ejemplos y documentación completa, visita los siguientes enlaces:

https://www.w3schools.com/js/default.asp
https://developer.mozilla.org/es/docs/Web/JavaScript
