CSS Display Property

La propiedad display especifica el tipo de caja de visualización utilizada para un elemento HTML.

.element {
  display: block;
}

CSS Position

La propiedad position especifica el tipo de método de posicionamiento utilizado para un elemento.

.element {
  position: relative;
  top: 20px;
  left: 15px;
}

CSS Flexbox

El módulo de caja flexible, comúnmente conocido como flexbox, es un modelo de diseño unidimensional.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
